def get_completion(prompt, model="gpt-3.5-turbo"):
    messages = [{"role": "user", "content": prompt}]
    response = openai.ChatCompletion.create(
        model=model,
        messages=messages,
        temperature=0, # this is the degree of randomness of the model's output
    )
    return response.choices[0].message["content"]

1/Prompting Principles

Principle 1: Write clear and specific instructions

  • Tactic 1: Use delimiters to clearly indicate distinct parts of the input
    • Delimiters can be any clear punctuation that separates the text from the rest of the promt:

      triple backticks ```, triple quotes """, triple dashes —-, angle brackets< >, XML tags<tag> </tag>, section titles Text:

  • Tactic 2: Ask for a structured output
    • JSON, HTML

      Provide them in JSON format with the following keys

  • Tactic 3: Check whether condition is satisfied
    • You will be provided with text delimited by triple quotes.
    • If …, then … . If …, then…
  • Tactic 4: "Few-shot" prompting
    • Your task is to answer in a consistent style

Principle 2: Give the model time to “think”

  • Tactic 1: Specify the steps required to complete a task
    • Perform the following actions: 1- , 2- , …
  • Tactic 2: Instruct the model to work out its own solution before rushing to a conclusion
    • Don't decide if the student's solution is correct until you have solved the problem yourself.

Model Limitations: Hallucinations

  • Sounds plausible but not true

2/Iterative Prompt Development

Issue 1: The text it generate is too long¶

  • Limit the number of words/sentences/characters.
    • Use at most 50 words.

Issue 2. Text focuses on the wrong details¶

  • Ask it to focus on the aspects that are relevant to the intended audience.
    • The description is intended for <client>, so should be <language feature> in nature and focus on <aspect>

Issue 3. Description needs a table of dimensions

  • Ask it to extract information and organize it in a table.
    • At the end of the description, include every <restriction> <target> in the <text>.
    • After the description, include a table that gives … The table should have <num> columns. In the first column, include the name of ... In the second column, include the measurements in <units>.
    • Give the table the title '<title>'.
    • Format everything as HTML
    • display HTML in python

def get_completion_from_messages(messages, model="gpt-3.5-turbo", temperature=0):
    response = openai.ChatCompletion.create(
        model=model,
        messages=messages, # put a list of messages: [user msg, sys msg, ...] as input instead of a single prompt
        temperature=temperature, # this is the degree of randomness of the model's output
    )
#     print(str(response.choices[0].message))
    return response.choices[0].message["content"]
